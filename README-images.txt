Instruções para imagens:

- Substitua estes arquivos colocando as imagens anexadas nesta pasta `assets/` com os nomes abaixo.

- `logo.png` — logo do ateliê (usado no cabeçalho e rodapé)
- `room.jpg` — foto do ambiente (hero)
- `room2.jpg` — imagem ilustrativa para a seção "Sobre"
- `map-placeholder.png` — imagem do mapa / localização (pode usar screenshot do Google Maps)

Se você preferir, mantenha os nomes e eu ajusto o HTML/CSS para outros nomes.
