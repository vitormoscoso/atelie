# Landing page — InspirArte

Esta é uma landing page estática baseada nas imagens que você forneceu. Coloque as imagens anexadas na pasta `assets/` com os nomes abaixo para que o layout funcione corretamente.

Arquivos criados:

- `index.html` — página principal
- `css/styles.css` — estilos
- `assets/README-images.txt` — instruções sobre imagens

Imagens recomendadas (arraste as imagens que você enviou para `assets/` com estes nomes):

- `logo.png` — logo (usado no cabeçalho e rodapé)
- `room.jpg` — foto principal do ateliê (hero)
- `room2.jpg` — foto para a seção "Sobre"
- `map-placeholder.png` — imagem do mapa / localização (ou print de mapa)

Como visualizar localmente (Windows PowerShell):

1. Abrir o arquivo diretamente (arquivo local):

```powershell
Start-Process index.html
```

2. Servidor simples com Python (recomendado para links, imagens e testing):

```powershell
python -m http.server 8000; Start-Process "http://localhost:8000"
```
